## Installation

learning-cats is available on scala 2.11, 2.12, 2.13 and scalajs.

Add the following to your build.sbt
```scala
libraryDependencies ++= Seq(
  "de.jds.edu" %% "learning-cats-core" % "0.0.1")
```
